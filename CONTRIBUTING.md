# 贡献指南

感谢你对灵枢 AI 掌纹溯源感兴趣！

## 如何参与

### 报告问题
1. [新建 Issue](https://github.com/YTZL2026/lingshu-ai-palm/issues)
2. 描述环境（操作系统、Python版本、浏览器）
3. 附上截图或错误日志

### 提交代码
1. Fork → 创建分支 → 修改 → 提交 → Pull Request
2. 提交规范：`feat:` / `fix:` / `docs:` / `refactor:`
3. 大的改动请先开 Issue 讨论

## 技术规范

- Python 3.10+，类型注解
- Flask 路由用装饰器模式
- 前端 JS 模块化，用 ES6+
- 注释用中文
- 敏感信息（API Key）永远不要提交

## 最需要的帮助

- 📱 Android 真机测试反馈
- 🍎 iOS Capacitor 适配
- 👅 舌诊模块开发
- 📊 体质判定准确率验证
- 🌐 英文/日文翻译

## 开源协议

本项目采用 AGPL-3.0。如要商用，请同样开源你的代码。
